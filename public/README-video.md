# Hero background video

Add your fullscreen hero clip as `public/hero-video.mp4`.

## Recommended specs

| Setting | Value |
| --- | --- |
| Format | MP4 (H.264) |
| Max file size | ~5 MB |
| Resolution | 1920×1080 |
| Duration | 8–15 seconds (seamless loop) |
| Audio | None (video plays muted) |

## Notes

- Until the file is added, the section shows the poster image (`/logo.png`).
- Keep motion subtle; heavy bitrate files slow first paint on mobile.
- Re-export if the loop has a visible jump at the cut point.
